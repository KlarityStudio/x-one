Easy User Tags adds a WordPress custom taxonomy called User Tags.

== Description ==

Easy User Tags lets you group your users by the 'User Tag' custom taxonomy and includes the ability to bulk add and remove tags from users as well as filter users by tag in the admin list view.


== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Easy User Tags'
3. Activate Easy User Tags from your Plugins page.

= From WordPress.org =

1. Download Easy User Tags.
2. Upload the 'easy-user-tags' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Easy User Tags from your Plugins page.
